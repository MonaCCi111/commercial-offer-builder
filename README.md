# Commercial Offer Builder

Утилита для автоматической генерации коммерческих предложений (КП) в формате Excel на основе неструктурированных данных с использованием ИИ.

Проект позволяет переводить сырой текст или запросы в структурированный формат и формировать готовый, красиво оформленный `.xlsx` документ, который уже оптимизирован для печати и экспорта в PDF.

## Особенности

- **ИИ-парсинг данных:** Преобразование произвольного текста или списков оборудования/услуг в структурированный JSON формат с распознаванием наименований, единиц измерения, количества и цен.
- **Генерация Excel-документов:** Автоматическое создание файла КП с использованием `openpyxl`.
- **Подсветка "сомнительных" позиций:** Если ИИ не уверен в цене или наличии позиции, строка в документе подсвечивается (колонка с примечаниями от ИИ исключается при экспорте в PDF).
- **Готовность к печати:** Сгенерированный Excel файл имеет заранее настроенные параметры страницы (книжная ориентация, масштабирование по ширине, заданная область печати), что обеспечивает корректный и красивый экспорт в PDF.
- **Поддержка логотипов:** Автоматическая вставка логотипа компании (при наличии файла `logo.png` в директории скрипта).

## Требования

- Python 3.8+
- Зависимости из `requirements.txt`

## Установка

1. Склонируйте репозиторий или скачайте файлы проекта.
2. Рекомендуется создать виртуальное окружение:
   ```bash
   python -m venv build-env
   ```
3. Активируйте виртуальное окружение:
   - На Windows: `build-env\Scripts\activate`
   - На Linux/macOS: `source build-env/bin/activate`
4. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

Основная логика запуска описана в `main.py`.

Пример генерации документа вручную (для тестов):

```python
import json
from excel_generator import OfferGenerator

# Пример данных, которые обычно возвращает ИИ модуль
data = [
    {
        "n": "Пример товара 1",
        "u": "шт",
        "q": 10,
        "p": 1500,
        "r": False,
        "rr": ""
    },
    {
        "n": "Товар без точной цены (требует проверки)",
        "u": "компл",
        "q": 1,
        "p": 0,
        "r": True,
        "rr": "Цена не указана в прайсе"
    }
]

generator = OfferGenerator()
generator.generate(json.dumps(data), output_path="my_offer.xlsx")
```

## Структура проекта

- `main.py` — Точка входа в приложение.
- `excel_generator.py` — Модуль для формирования и стилизации Excel-файла (использует `openpyxl`).
- `ai_engine.py` — Модуль взаимодействия с AI для парсинга входящих данных в JSON.
- `check_models.py` — Скрипт для тестирования/проверки доступных ИИ моделей.
- `requirements.txt` — Список зависимостей Python.
- `logo.png` — (Опционально) Логотип, который будет помещен в шапку генерируемого документа.

## Экспорт в PDF

Документы, созданные с помощью `OfferGenerator`, автоматически получают корректную "Область печати" (Print Area). При открытии файла в Excel и выборе "Сохранить как -> PDF", технические столбцы (например, примечания ИИ) будут скрыты, а таблица аккуратно впишется по ширине листа А4.
