# Commercial Offer Builder (Генератор КП)

Утилита для автоматической генерации коммерческих предложений (КП) в формате Excel на основе неструктурированных данных с использованием нейросетей Google Gemini. 

Проект позволяет загружать сырые документы (PDF-файлы, Excel-таблицы, фотографии и сканы чеков/накладных) и формировать готовый, красиво оформленный `.xlsx` документ со сметой, который оптимизирован для печати и экспорта в PDF.

## Особенности

- **Графический интерфейс (GUI):** Интуитивно понятный интерфейс на базе `CustomTkinter` для выбора файлов и отслеживания прогресса обработки.
- **Двухэтапный ИИ-парсинг:** 
  - *Анализатор:* Предварительно изучает структуру документа (колонки, форматы цен, аномалии) и составляет шпаргалку.
  - *Парсер:* Преобразует документ в структурированный JSON с распознаванием наименований, единиц измерения, количества и цен на основе инструкций Анализатора.
- **Поддержка разных моделей Gemini:** Автоматический выбор оптимальной модели (например, `gemini-3.1-flash-lite-preview` для текстов/таблиц и `gemini-3-flash-preview` для изображений).
- **Поддержка множества форматов:** Обработка `.pdf`, `.xlsx`, `.xls`, `.jpg`, `.jpeg`, `.png`.
- **Генерация Excel-документов:** Автоматическое создание файла КП с использованием `openpyxl`. Подсчет итоговой суммы для каждой позиции.
- **Подсветка "сомнительных" позиций:** Если ИИ не уверен в цене, названии или наличии позиции (например, текст размыт), строка в документе подсвечивается. Техническая колонка с примечаниями от ИИ исключается при экспорте в PDF.
- **Готовность к печати:** Сгенерированный Excel-файл имеет заранее настроенную область печати (Print Area) и масштабирование по ширине А4.
- **Поддержка логотипов:** Автоматическая вставка логотипа компании (при наличии файла `logo.png` в директории скрипта).

## Требования

- Python 3.8+
- API-ключ Google Gemini
- Вспомогательный Worker URL (опционально, в зависимости от вашей архитектуры обхода блокировок)

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/MonaCCi111/commercial-offer-builder.git
   cd commercial-offer-builder
   ```
2. Рекомендуется создать виртуальное окружение:
   ```bash
   python -m venv build-env
   ```
3. Активируйте виртуальное окружение:
   - На Windows: `build-env\Scripts\activate`
   - На Linux/macOS: `source build-env/bin/activate`
4. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Настройка (Конфигурация)

При первом запуске программа автоматически создаст файлы настроек: `config.json`, `prompt.txt` и `analyzer_prompt.txt` в папке со скриптом.

Вам необходимо открыть `config.json` и указать свой API-ключ Gemini:

```json
{
    "GEMINI_API_KEY": "ВАШ_КЛЮЧ_GEMINI",
    "WORKER_URL": "ВАШ_WORKER_URL_ЕСЛИ_ЕСТЬ",
    "MODEL_LITE": "gemini-3.1-flash-lite-preview",
    "MODEL_VISION": "gemini-3-flash-preview",
    "TIMEOUT_MS": 60000,
    "EXCEL_CHUNK_SIZE": 100,
    "PDF_CHUNK_SIZE": 3,
    "EXCEL_HEAD": 100,
    "PDF_HEAD": 2
}
```

Вы также можете кастомизировать поведение ИИ, отредактировав файлы `prompt.txt` (инструкция для извлечения позиций) и `analyzer_prompt.txt` (инструкция для предварительного анализа таблиц/страниц).

## Использование

1. Запустите приложение:
   ```bash
   python main.py
   ```
2. В открывшемся окне нажмите **"Выбрать файлы"** и выберите один или несколько документов (PDF, Excel, картинки).
3. Нажмите **"Сгенерировать КП"**.
4. Дождитесь окончания обработки (прогресс и статус будут отображаться в окне). 
5. По завершении готовый файл будет сохранен в папку `Документы/КП_Тепломир`, и папка автоматически откроется в проводнике.

## Структура проекта

- `main.py` — Точка входа в приложение, отрисовка GUI на `CustomTkinter`.
- `config.py` — Управление конфигурацией (создание и чтение `config.json` и `.txt` промптов).
- `ai_engine.py` — Взаимодействие с API Gemini (отправка чанков документов, получение структурированного ответа).
- `excel_generator.py` — Формирование и стилизация Excel-файла (использует `openpyxl`).
- `check_models.py` — Скрипт для тестирования доступных моделей Gemini.
- `requirements.txt` — Список зависимостей Python.

## Экспорт в PDF

Документы, созданные генератором, автоматически получают корректную "Область печати" (Print Area). 
При открытии сгенерированного файла в Excel и выборе **"Сохранить как -> PDF"**, технические столбцы (например, примечания от ИИ) будут скрыты, а таблица со сметой аккуратно впишется по ширине на лист формата А4.
